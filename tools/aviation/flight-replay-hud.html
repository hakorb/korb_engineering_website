[?] HOW IT WORKS

Drop in any supported flight log — GPX, KML/KMZ, DJI Airdata CSV, or a generic CSV with latitude, longitude, altitude, and timestamp columns. Column headers are fuzzy-matched, so lat, Latitude, OSD.latitude, etc. all work. The HUD animates in real-time, interpolates between log samples, and optionally reads heading, speed, roll, pitch, yaw, and battery. If no heading is present, it's derived from bearing between samples. If no speed is present, it's computed from distance/time. Everything happens in your browser — no uploads, no tracking. Big aerospace energy, zero server dependency.
