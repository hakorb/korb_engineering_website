How PAPI Works

  • 4 light units mounted beside the runway (typically left side, abeam touchdown zone)
  • Each unit projects a split beam: red below a cutoff angle, white above
  • Standard 3° glide path uses 2.5° / 2.833° / 3.167° / 3.5° cutoffs (inner to outer)
  • APAPI (Abbreviated PAPI) uses 2 units and is common on shorter / GA-category runways
  • Effective range: ~3.4 NM day, up to 20 NM at night
  • Each unit must be set within ±3 arc-minutes (±0.05°) of nominal per FAA AC 150/5345-28
  • System tolerance is ±1/4° total; periodic flight inspection verifies calibration

About This Tool

Built for pilots rehearsing glidepath reads, airport engineers prepping inspection flights, and PMs who need a screenshot-worthy PAPI visual for a briefing. All computation runs locally; inspection logs live in your browser's localStorage under korb_papi_.

Nominal 3° glide path is assumed throughout. Override individual unit angles on the Indicator tab to match a non-standard slope (common at high-altitude or obstacle-constrained airports).
