Photogrammetry Studio

A browser-based, open-source photogrammetry pipeline inspired by Meshroom. Takes a set of photos and reconstructs a 3D model using a simplified Structure-from-Motion pipeline.

Pipeline

  • CameraInit — Load images, extract metadata.
  • FeatureExtraction — Harris corner detection + intensity-patch descriptors.
  • ImageMatching — Determine which image pairs share content.
  • FeatureMatching — Brute-force descriptor matching with ratio test.
  • StructureFromMotion — Estimate camera poses on a ring, triangulate points.
  • DepthMap — Densify sparse cloud via interpolation.
  • Meshing — Convex-hull meshing (ball-pivoting approximation).
  • Texturing — Per-face color projected from source images.

Honest Disclaimer

This is a simplified/approximate pipeline designed to run entirely in a browser. Real SfM solvers (COLMAP, Meshroom, OpenMVG) use bundle adjustment, RANSAC, Poisson surface reconstruction, and MVS — none of which fit in one HTML file. For real engineering-grade reconstructions, use dedicated desktop software.

Tips

  • Use 15-30 photos orbiting an object with ~10-20° between shots.
  • Avoid shiny / transparent / texture-less subjects.
  • Hit Demo Mode to explore the UI without uploading.
