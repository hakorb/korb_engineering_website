Reference & Formulas

Rational Method
Q = C x i x A
Q = Peak runoff rate (CFS)
C = Runoff coefficient (dimensionless, 0-1)
i = Rainfall intensity (in/hr) at duration = Tc
A = Drainage area (acres)

Valid for drainage areas up to ~200 acres. Assumes uniform rainfall over the area and that peak runoff occurs when the entire area is contributing.
Manning's Equation (Full Pipe)
Q = (1.486/n) x A x R2/3 x S1/2
For circular pipe flowing full:
Q = (0.4632/n) x D8/3 x S1/2

Q = Flow capacity (CFS)
n = Manning's roughness coefficient
D = Pipe diameter (ft)
S = Pipe slope (ft/ft)
A = Cross-sectional area
R = Hydraulic radius = D/4 for full pipe
Velocity Limits
Minimum: 2.5 fps (self-cleaning velocity to prevent sediment buildup)
Maximum RCP/HDPE/PVC/DI: 15 fps
Maximum CMP: 10 fps (erosion/abrasion limit)

Optimal range: 3-8 fps for most applications.
