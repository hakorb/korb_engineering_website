Data Sources & Methodology

1. USAspending.gov

POST https://api.usaspending.gov/api/v2/search/spending_by_award/ with a recipient_search_text filter. Returns federal contract awards (PIID, agency, amount obligated, period of performance, place of performance, type code). CORS-enabled. No API key.

2. SEC EDGAR Full-Text Search

GET https://efts.sec.gov/LATEST/search-index?q="Name"&forms=10-K,10-Q,8-K. Surfaces material litigation and going-concern language for publicly listed contractors.

3. CourtListener (RECAP)

GET https://www.courtlistener.com/api/rest/v3/search/?q=<name>&type=r. Federal court filings — bankruptcies, civil suits, mechanic's liens.

4. Wikipedia API

GET https://en.wikipedia.org/w/api.php?origin=*&action=query&list=search&srsearch=<name>. Notability signal.

5. GitHub Search

GET https://api.github.com/search/repositories?q=<name>. Optional digital-maturity signal.

6. OSHA (manual paste)

OSHA does not expose a CORS-friendly JSON API. The tool launches Establishment Search and parses the text the user pastes back, scanning for inspection IDs, standards (e.g. 1926.501), penalty totals, fatalities and hospitalizations.

7. Launch links only

SAM.gov, OpenCorporates, BBB, Texas/Oklahoma/New Mexico Secretary of State, state contractor license boards, D&B, PACER, FEC, LinkedIn, BuzzFile.

Disclaimer

This tool provides directional signals built from public data. It is not a credit report, not a legal opinion, and not a substitute for a Dun & Bradstreet or Experian Business report. Always pull primary documents and have counsel review before any award decision.

Privacy

Everything happens in your browser. Lookups, watchlist, and OSHA notes are stored only in this device's localStorage under korb_contractor_bgcheck_*. Nothing is sent to Korb Engineering servers.
