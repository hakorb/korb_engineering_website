// About This Tool

FE/PE Civil Practice Bank is a free, single-file practice environment built by Korb Engineering. It replaces $200+ per-year subscription services with a local, offline-capable, open-format study tool.


Content: 80+ original multiple-choice problems covering the 12 official FE Civil topic areas, each with full worked solutions, formula references, and NCEES topic-area mapping.


How to use:

  • Quiz Mode — build custom sessions by topic, difficulty, and length
  • Study Mode — browse the bank with solutions always visible
  • Flashcards — drill formulas until they're automatic
  • Exam Simulator — a full 110-question, 5h 20m dry run
  • Progress — localStorage-backed tracking; exportable JSON

Disclaimer: This is an unofficial study aid. The NCEES FE Reference Handbook is the authoritative source for formulas used on the exam. Pass rates and topic weights reflect publicly available data and may change.
