Field Name Type Values / Notes Required
FAA Airport ID Text ICAO (4-char) or IATA (3-char) · e.g., KORD, KLGB Yes
Contract / AIP Grant Number Text e.g., 3-25-0042-025-2026 Yes
Submittal Round Dropdown 30% · 60% · 90% · 100% Final · Addendum Yes
PCI Score Number 0–100 integer · populate after E3 analysis complete No
PCN Value Number Decimal · per ICAO format (e.g., 55/F/B/X/T) — store numeric portion only No
Survey Type Dropdown PCI · PCN · FWD · Photogrammetry · FCA · Geotech · Site Survey Yes

Sprint Structure
Generic 4-sprint cadence (2-week sprints) for an 8-week airport pavement survey project
Note: Create sprints under Scrum → Sprints. Velocity baseline: ~40 story points per sprint for a 3-person team. Story point scale: Fibonacci (1, 2, 3, 5, 8, 13, 21).
Sprint 1
Weeks 1–2
Mobilize & Collect
Execute contract, complete pre-field logistics, and mobilize for field survey. Begin PCI distress survey.
US-1.1 · US-1.2 · US-1.3
US-1.4 · US-1.5
US-2.1 (start)
Focus: E1 Initiation + E2 Field Work
▶ Milestone: Field mobilization complete; crew on-site; PCI survey ≥50% complete
Sprint 2
Weeks 3–4
Complete Field & Begin Analysis
Complete all field data collection (PCI, FWD, drone, FCA) and begin photogrammetry processing.
US-2.1 (complete)
US-2.2 · US-2.3 · US-2.4
US-3.3 (start)
Focus: E2 Field Work + E3 Analysis start
▶ Milestone: 100% field data collected; raw data backed up; FIELD-COMPLETE tag applied
Sprint 3
Weeks 5–6
Analysis & Draft Deliverables
Complete all analyses (PCI/PCN/backcalculation/ortho) and produce draft deliverables for internal QC.
US-3.1 · US-3.2
US-3.3 (complete) · US-3.5
US-4.1 (draft) · US-4.4
Focus: E3 Analysis + E4 Production start
▶ Milestone: Internal QC sign-off; all QC-REQUIRED stories cleared

Weeks 7–8
