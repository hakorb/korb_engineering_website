    Walk-Scan Mode

    Stand in one corner of the room, aim the device along the first wall, and tap Start Camera. At each corner, tap Mark Corner — the device's compass captures the bearing. Tap Set Wall Length to enter the measured distance back to the previous corner (feet).

    Walk Mode requires permission for camera, location, and device orientation. iOS Safari will prompt you on the first tap.

    Export & share

    The .obj file contains floor + walls + roof as a triangle mesh. The .json file bundles corner coordinates, walls, and a georeferencing block (lat, lon, heading) so you can re-import into CAD or GIS with a known origin.
