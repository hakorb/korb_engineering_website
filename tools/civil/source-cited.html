[ P05 ] NOTES & LIMITATIONS
• All fetching and parsing happens client-side via api.allorigins.win (a public CORS proxy). If that service is down, edit the fields manually.
• Auto-parse looks for: OpenGraph tags, JSON-LD metadata (schema.org), author meta tags, and article time/date tags. Accuracy depends on the source site's markup.
• The tool handles web pages well. Journal articles and books often need manual cleanup.
• This is an authoring aid, not a replacement for your journal/publisher's style checker. Always review citations before submitting.
