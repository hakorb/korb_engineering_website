Danger Zone Irreversible
Removing a credential deletes the stored reference on this device only. The secure-enclave key on the originating hardware must also be deleted via the OS’s Passkey manager if you want to fully revoke it.
Honest limitation This tool uses WebAuthn as a local-only biometric gate, not a full remote attestation. Challenges are generated and verified client-side via crypto.getRandomValues(). Because there is no authoritative server, a sufficiently motivated attacker with full local storage access could bypass the check. Treat this as a convenient UX shortcut on top of the existing password — not a substitute for server-enforced authentication.
