Cron Cheat Sheet

 ┌──────────── minute       (0 - 59)
 │ ┌────────── hour         (0 - 23)
 │ │ ┌──────── day of month (1 - 31)
 │ │ │ ┌────── month        (1 - 12)  or JAN-DEC
 │ │ │ │ ┌──── day of week  (0 - 6)   or SUN-SAT, 0/7 = Sunday
 │ │ │ │ │
 * * * * *

Operators

* — every value in the range
*/N — every N values (step). Example */15 in minute field = :00 :15 :30 :45
A-B — range, inclusive. Example 1-5 in dow = Mon-Fri
A,B,C — list. Example 0,30 in minute = top & bottom of the hour
A-B/N — step within range. Example 2-22/4 in hour

Shortcuts

@yearly (@annually) = 0 0 1 1 *
@monthly = 0 0 1 * *
@weekly = 0 0 * * 0
@daily (@midnight) = 0 0 * * *
@hourly = 0 * * * *
