Data Management

Raw Storage Keys

All data lives under korb_game_deal_* in localStorage.

  • korb_game_deal_library — games array
  • korb_game_deal_settings — currency rates & prefs
  • korb_game_deal_version — schema version

How It Works

Tracking Games

Add games via the Add Game tab or the Catalog tab. Each game stores MSRP, current price, target price, store, platform, tags, and notes.

When current < target, the card gets a green DEAL badge and appears in the "On Sale" kanban column automatically if you enable auto-flag.

Price History & Charts

Each manual price update appends to the game's price history. Click "History" on any card to see the trend line (pure Canvas, no libraries) and the raw price log.

The chart y-axis auto-scales; x-axis shows chronological update order.

Store Search Links

Every card generates search URLs (not API calls) for Steam, GOG, and Epic Games Store. Tapping a link opens the store's native search page for that title.

Kanban Backlog

Four columns: Wishlist → On Sale → Owned → Completed. Drag on desktop or use the ► buttons on cards. Great for managing your personal backlog pipeline.

Currency Conversion

Set your base display currency and manual exchange rates in Settings. Since stores don't offer free rate APIs, you enter rates manually — update them when you notice drift.

All prices normalize internally to USD and display in the selected base.

Import / Export

Export produces a JSON file of your entire library + settings. Import merges or replaces (you'll be asked). Great for device-to-device moves or backups.

Disclaimer

This is a personal tracker. It never contacts any store APIs or scrapes prices — you update prices manually when you spot deals. Catalog MSRPs are placeholder defaults and may differ from actual store pricing.
