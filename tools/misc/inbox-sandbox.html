SPF / DKIM / DMARC Explainer

Educational reference for the three major email authentication standards. Not a live DNS lookup - this is a teaching tool that explains what each record means.

SPF (Sender Policy Framework)

SPF is a TXT DNS record at the sending domain listing which servers are authorized to send mail for it. Receiving servers check the envelope-from (MAIL FROM) against this list. Defined in RFC 7208.
(not analyzed)

DKIM (DomainKeys Identified Mail)

DKIM cryptographically signs outgoing messages. The public key lives at selector._domainkey.domain.tld. Receivers verify the signature proves the message wasn't altered in transit. Defined in RFC 6376.
(not analyzed)

DMARC (Domain-based Message Authentication)

DMARC ties SPF/DKIM together with alignment rules and a policy telling receivers what to do with failing mail (none, quarantine, or reject). Defined in RFC 7489.
(not analyzed)
