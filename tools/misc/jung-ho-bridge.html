사용 방법 · How To Use
• 🎤 마이크 버튼: 말하면 자동 입력
• 📋 빠른 문구: 탭하면 바로 번역 탭에 로드
• 🔊 모든 텍스트: 스피커 버튼으로 재생 가능
• 📲 홈 화면 추가 (Chrome): 메뉴 → 홈 화면에 추가

• 🎤 Mic button: speak to auto-input
• 📋 Quick Phrases: tap card to load into Translate
• 🔊 Any text: tap speaker button to hear it
• 📲 Add to Home Screen (Chrome): menu → Add to Home Screen
방언 · Dialect Note
이 앱은 표준어를 사용합니다 — 경상도 사투리를 사용하는 분도 완전히 이해할 수 있습니다.
This app uses Standard Korean (표준어) — fully understood by speakers of all regional dialects including Gyeongsang (경상도).
